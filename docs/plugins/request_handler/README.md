# request_handler（申请管理）

好友申请与入群申请提醒与审批：私聊提醒号主，支持快捷处理、按号码处理、批量处理与自动同意开关。

## 用户命令

| 口令 | 场景 | 说明 |
| --- | --- | --- |
| 查看好友申请 / 查看入群申请 | 私聊 | 查看当前待处理申请（兼容旧口令“查看入群邀请”） |
| 同意 / 拒绝 | 私聊 | 处理最新提醒，或引用某条提醒只处理该条 |
| 同意好友 \<QQ\> / 拒绝好友 \<QQ\> | 私聊 | 按 QQ 处理单条好友申请 |
| 同意入群 \<群号\> / 拒绝入群 \<群号\> | 私聊 | 按群号处理单条入群申请 |
| 同意所有好友/入群 / 拒绝所有好友/入群 | 私聊 | 批量处理当前列表 |
| 查看自动同意 / 开启或关闭自动同意好友 / 入群 | 私聊 | 查看或切换自动同意 |

## 命令权限

审批命令默认按 cmd_perm 配置控制，代码默认等级为 **bot_moderator**；实际以 WebUI / cmd_perm 覆盖为准。

## 配置

[`config.py`](../../../src/plugins/request_handler/config.py)：`request_handler_poll_doubt_friends` 等。

## 排障

| 现象 | 处理 |
| --- | --- |
| 无提醒 | 插件是否被 `牛牛关闭`；号主能否收私聊 |
| 同意/拒绝无效 | 快捷提醒已过期（约 7 天）；先重新查看列表，或改用带 QQ/群号命令 |
