# dream（牛牛做梦）

> **官方扩展**：`pallas-plugin-dream`（`uv sync --extra plugins-dream`）

做梦期间推送梦话：可收到他群漂流、历史梦、图片等；醉酒时更频繁。

## 用户命令

| 口令 | 场景 | 说明 |
| --- | --- | --- |
| 牛牛做梦 | 群内 | 进入做梦约 5～15 分钟 |
| 牛牛醒梦 / 牛牛别做梦 | 群内 | 结束做梦 |
| 牛牛醒一醒 | 群内 | 醒酒时一并结束做梦（见 [drink](./drink/README.md)） |

## 命令权限

| 命令 ID | 默认等级 |
| --- | --- |
| `dream.ban_cleanup` | staff |

（与复读「不可以」共用清理梦库，见帮助详情。）

## 配置

WebUI **插件 → 牛牛做梦** 或 [`config.py`](../../../src/plugins/dream/config.py)（梦话间隔、保留天数等）。

## 排障

| 现象 | 处理 |
| --- | --- |
| 无梦话 | 确认已发「牛牛做梦」；他群无人做梦则无漂流 |
| 内容过多 | 调小保留天数或联系维护者清理 |
