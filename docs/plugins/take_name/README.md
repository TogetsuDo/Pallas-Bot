# take_name（自动夺舍）

牛牛按极低概率 **随机同步群内某成员的群名片** 到自己，并 **戳一戳** 对方；在 **醉酒** 且牛牛为 **群管理员** 时，有小概率把对方名片改成固定名。当被「夺舍」对象 **修改自己的群名片** 时，牛牛会 **再次同步** 对方名片到自己。

## 依赖

- **复读插件** `MessageStore`：用于从各群抽取随机消息以选取目标成员。
- **喝酒状态**：醉酒夺舍依赖 `BotConfig` 醉酒度与群管权限。

## 配置

无独立 `config.py`，每分钟约 `0.002` 概率尝试改名。

## 排障

| 现象 | 说明 |
|------|------|
| 从不改名 | 概率极低属正常；确认群未睡眠、Bot 仍在群且有权改名片。 |
| 改名片失败 | 无群管理员权限时无法改他人名片；协议端可能拒绝 `set_group_card`。 |

实现见 [`src/plugins/take_name/__init__.py`](../../../src/plugins/take_name/__init__.py)。
