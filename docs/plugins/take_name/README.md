# take_name（自动夺舍）

定时随机改牛牛群名片；醉酒时可能同步改被模仿群友名片。

## 用户命令

| 触发 | 场景 | 说明 |
| --- | --- | --- |
| 定时任务 | 自动 | 模仿群友名片并戳一戳 |
| 牛牛醉酒 | 自动 | 可能「夺舍」改名 |

## 命令权限

无用户口令。

## 配置

[`config.py`](../../../src/plugins/take_name/config.py)。依赖 `repeater` 的 `MessageStore` 取随机语料。

## 排障

| 现象 | 处理 |
| --- | --- |
| 从不改名 | 概率较低属正常；查插件是否被群关闭 |
| API 失败 | 牛牛需有改名片权限 |
