# roulette（牛牛轮盘）

群内 **管理员** 发起轮盘（踢人或禁言模式），成员 `牛牛开枪` 参与；命中可能 **被踢** 或 **被禁言**。`牛牛救一下` / `牛牛补一枪` 可解禁或追加惩罚，均有 **失败/炸膛** 概率，**醉酒** 后部分效果增强。与 `drink` 插件的醉酒状态联动。

## 权限与风险

- 启动「牛牛轮盘」及子命令、以及游戏中「牛牛开枪」：均要求 **牛牛账号在该群为群主或管理员**（否则规则不通过）；普通群员无法代开。
- **会真实踢人、禁言**，请在允许的群使用并做好群规说明。

## 配置

玩法数值主要在 [`src/plugins/roulette/config.py`](../../../src/plugins/roulette/config.py)（`RESCUE_CFG`、`JUDGMENT_CFG`、`SHOT_CFG` 等 dataclass），可按需调整概率与文案。

## 排障

| 现象 | 说明 |
|------|------|
| 操作无效果 | 确认 Bot 在群内有 **踢人/禁言** 权限；游戏是否已超时关闭（内置超时，见源码 `timeout`）。 |
| 误踢误禁 | 调低命中概率或关闭插件；事后依赖群主/超管人工处理。 |

实现见 [`src/plugins/roulette/__init__.py`](../../../src/plugins/roulette/__init__.py)。
