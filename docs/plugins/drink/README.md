# drink（牛牛喝酒）

群内发送 **牛牛喝酒** / **牛牛干杯** / **牛牛继续喝** 增加醉酒度，可能触发 **睡着**（长时间不响应类行为由其他插件配合）；**牛牛醒一醒** / **牛牛别喝了** 立即清醒。每日定时任务会清空全局醉酒数据（见源码）。

## 关联

醉酒状态由 `BotConfig` 持久化，影响 **复读**、**轮盘**、**夺舍**、**酒后聊天** 等多个插件的行为。

## 配置

无独立环境变量配置文件，冷却等逻辑在 [`src/plugins/drink/__init__.py`](../../../src/plugins/drink/__init__.py) 内与 `BotConfig` 交互。

## 排障

| 现象 | 说明 |
|------|------|
| 喝酒无反应 | 群冷却内可能静默；查看是否被风控或 `ActionFailed`。 |
| 一直不醒 | 用 `牛牛醒一醒`；或等待定时 `fully_sober_up`。 |

实现见 [`src/plugins/drink/__init__.py`](../../../src/plugins/drink/__init__.py)。
