# 口令与功能

群里发下面这些（或 @ 牛）即可触发。  
**最全列表**以群里 **牛牛帮助** 的图为准（会标「何人可用」）。

::: tip 权限三个词
| 词 | 意思 |
| --- | --- |
| 超管 | `pallas.toml` 里 `superusers` |
| 号主 | 该牛管理员列表里的 QQ |
| 群管 | QQ 群管理员 |
:::

## 开箱就有（core）

| 你发 | 牛会 |
| --- | --- |
| `牛牛帮助` | 帮助图；群管可关单群插件 |
| `牛牛` | 打招呼（多牛同群可能齐回） |
| `牛牛喝酒` / `醒一醒` | 喝酒玩法 |
| `牛牛轮盘` | 轮盘；`牛牛救一下` / `牛牛补一枪` |
| （自然接话） | 学群友说话后复读接话 |

## 要先装官方插件

| 你发 | 要先装 |
| --- | --- |
| 决斗、八角笼 | `pallas-plugin-duel` |
| 谁是卧底 | `pallas-plugin-who-is-spy` |
| `牛牛做梦` | `pallas-plugin-dream` |
| MAA 远控 | `pallas-plugin-maa` |

步骤：[安装官方插件](install-extensions.md)。

## 要另起 AI

| 你发 | 需要 |
| --- | --- |
| `牛牛唱歌` / 点歌 | [Pallas-Bot-AI](https://github.com/PallasBot/Pallas-Bot-AI) + 扩展 |
| `牛牛画画` | 同上 + draw |
| 随时 @ 闲聊 | 控制台打开 LLM；见 [AI 扩展](ai.md) |

可先发 `牛牛连通` 测 Bot 能不能连上 AI。

## 管理向（网页）

| 目的 | 去哪 |
| --- | --- |
| 改配置 | `http://<主机>:8088/pallas/` |
| 扫码上 QQ | `/protocol/console/` |
| 关插件 | 控制台 **插件** |

详见 [网页控制台](web-console.md)。
