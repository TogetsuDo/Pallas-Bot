# 使用指南

给 **已经跑起来的站长 / 群管**：日常在网页和群里怎么用。

还没连 QQ？先看 [五分钟跑起来](../guide/quickstart.md) 和 [连接协议端](../guide/connect-qq.md)。

## 三步上手

1. 打开 `http://<主机>:8088/pallas/`（端口以 `pallas.toml` 为准）
2. 用启动日志里的口令登录
3. 群里发 **`牛牛帮助`**，对照图里的功能

面板说明：[网页控制台](../guide/web-console.md)。

## 网页里常做的事

| 你想… | 操作 |
| --- | --- |
| 装决斗、MAA | **插件商店** → 安装 → 重启 |
| 改画画 / 唱歌 / AI | **通用配置** |
| 改谁能用某口令 | **命令权限** |
| 关一只牛的所有玩法 | **插件** → **全实例禁用** |
| 看为什么没反应 | **运行日志**、**日志报错** |
| 加 NapCat 账号 | **协议端实例** 或 `/protocol/console/` |
| 备份 | **数据库** |

保存后写入 `data/pallas_config/webui.json`。

## 群里常用口令

完整列表以 **牛牛帮助** 为准。摘要：[口令与功能](../guide/usage.md)。

| 口令 | 作用 |
| --- | --- |
| 牛牛帮助 | 功能总览 |
| 牛牛连通 | 测外部服务（含 AI） |
| 牛牛喝酒 / 醒一醒 | 喝酒状态 |
| 牛牛轮盘 | 轮盘玩法 |

## 排障

| 现象 | 先看 |
| --- | --- |
| 群里完全没反应 | 协议端是否在线；牛是否在群；[FAQ](../FAQ.md) |
| 某功能没有 | 插件是否安装；帮助图里是否被关掉 |
| AI 不可用 | 控制台 AI 地址；`牛牛连通` |
| 忘记控制台口令 | [FAQ](../FAQ.md) |
