# pallas-plugin-bot-status

Pallas-Bot 4.0 官方扩展：**牛牛状态**（在吗、报数、离线邮件）。

## 安装

```bash
uv sync --extra plugins-bot-status
```

## 功能

| 口令 | 说明 |
| --- | --- |
| 牛牛在吗 | 查在线/离线 |
| 牛牛报数 / 牛牛出列 | 群内依次报到 |
| 测试邮件 | 验证 SMTP |

配置见 WebUI **插件 → 牛牛状态**。
